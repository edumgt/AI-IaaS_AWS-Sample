# Chapter 03. AWS CLI 실무 사용법

## 챕터 개요
aws configure, profile, region, output, JMESPath 쿼리

## 학습 목표
- CLI로 인증 상태 점검 및 반복 작업 자동화
- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론
- named profile 운영
sts/s3/iam 핵심 명령
--query와 --output table

## 실습 커리큘럼
- 1) profile 2개 생성(dev/prod)
2) 현재 계정 식별자 조회
3) bucket 목록 테이블 출력

## 산출물
- 실행 명령어와 결과 로그
- 실패 원인/개선안을 포함한 회고 노트

## 완료 기준
- 이론 질문 5개 이상에 답변 가능
- 실습 3개를 독립적으로 재현 가능
- 다음 챕터 선수 조건 충족
