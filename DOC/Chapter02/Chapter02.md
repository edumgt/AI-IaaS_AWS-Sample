# Chapter 02. IAM 심화와 권한 설계

## 챕터 개요
정책(JSON), 사용자/그룹/역할(Role), 최소권한 모델

## 학습 목표
- Policy 문법을 읽고 필요한 권한만 부여
- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론
- Allow/Deny 평가 순서
Managed vs Inline Policy
Access Analyzer 기초

## 실습 커리큘럼
- 1) S3 전용 읽기 정책 작성
2) Rekognition 실행 정책 작성
3) 권한 시뮬레이터 검증

## 산출물
- 실행 명령어와 결과 로그
- 실패 원인/개선안을 포함한 회고 노트

## 완료 기준
- 이론 질문 5개 이상에 답변 가능
- 실습 3개를 독립적으로 재현 가능
- 다음 챕터 선수 조건 충족
