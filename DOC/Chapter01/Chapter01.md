# Chapter 01. AWS 기초와 계정 보안

- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론

리전 선택 전략

2) Billing Alert 생성
3) IAM Admin 사용자 생성 및 로그인 테스트

## 산출물
