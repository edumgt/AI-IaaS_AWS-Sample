# Chapter 07. Rekognition 실전 API

## 챕터 개요
S3 객체 기반 분석, 비동기 작업, 오류 처리

## 학습 목표
- 실무용 파이프라인 형태로 확장
- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론
- 입력 검증
재시도 전략
예외 케이스 대응

## 실습 커리큘럼
- 1) 여러 이미지 배치 분석
2) 실패 파일 목록 분리
3) 실행 리포트 작성

## 산출물
- 실행 명령어와 결과 로그
- 실패 원인/개선안을 포함한 회고 노트

## 완료 기준
- 이론 질문 5개 이상에 답변 가능
- 실습 3개를 독립적으로 재현 가능
- 다음 챕터 선수 조건 충족
