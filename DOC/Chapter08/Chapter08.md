# Chapter 08. 운영 자동화와 배치 스크립트

## 챕터 개요
bash 스크립팅, aws s3 sync/cp/rm, idempotent 처리

## 학습 목표
- CLI 자동화로 반복 작업 시간 단축
- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론
- set -euo pipefail
파라미터화/로그파일
크론/CI 연동

## 실습 커리큘럼
- 1) 일괄 업로드 스크립트 실행
2) dry-run으로 삭제 검증
3) 보고서 파일 생성

## 산출물
- 실행 명령어와 결과 로그
- 실패 원인/개선안을 포함한 회고 노트

## 완료 기준
- 이론 질문 5개 이상에 답변 가능
- 실습 3개를 독립적으로 재현 가능
- 다음 챕터 선수 조건 충족
