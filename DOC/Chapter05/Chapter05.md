# Chapter 05. Node.js와 AWS SDK 연동

## 챕터 개요
Node.js 런타임, npm, dotenv, aws-sdk(v2)

## 학습 목표
- 자격증명 로딩 순서와 코드 구조 이해
- 학습 결과를 문서화하고 재현 가능한 실습 절차를 확보

## 핵심 이론
- 프로젝트 구조 분석
환경변수 관리
에러 로그 읽는 방법

## 실습 커리큘럼
- 1) npm install
2) .env 구성
3) extract/compare/upload 스크립트 실행

## 산출물
- 실행 명령어와 결과 로그
- 실패 원인/개선안을 포함한 회고 노트
